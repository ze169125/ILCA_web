Instale como app no celular pra usar 100% offline.

Android — APK offline

DETECTADO
⬇ Baixar APK
Versão 1.4 · 2.9 MB · funciona 100% offline desde a instalação
  1. Baixa o arquivo (botão acima)
  2. Abre o arquivo .apk via Notificações ou Arquivos
  3. Sistema pode pedir "Permitir instalar deste app" — autoriza
  4. Toca Instalar
  5. Permite GPS quando o app pedir
  6. Pode ativar modo avião e testar — funciona offline normalmente

iPhone / iPad

DETECTADO

Não tem APK pra iOS (Apple não permite sideload). Instale como PWA pela tela inicial:

  1. Abre ze169125.github.io/ILCA_web/setup.html no Safari (não Chrome)
  2. Toca no botão Compartilhar (quadrado com seta)
  3. Rola e toca em "Adicionar à Tela de Início"
  4. Confirma — ícone amarelo aparece na home
  5. Abre uma vez online pra cachear, depois funciona offline
